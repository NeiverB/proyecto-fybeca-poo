# Sistema de Gestión Farmacéutica Fybeca

## Descripción

Proyecto Integrador desarrollado para la asignatura Programación Orientada a Objetos con Java.

El sistema permite gestionar clientes, medicamentos, ventas, facturas, inventario y proveedores de una farmacia inspirada en Fybeca. Además, incorpora indicadores de transformación digital y madurez digital para apoyar la toma de decisiones empresariales.
